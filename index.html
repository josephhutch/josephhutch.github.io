Projects

Franken Phone

Email through phone    •    August 2016

Rings when you get a new email. Reads it to you when you pick up. Want to send an email? Select a contact and speak your message into the phone. Hang up to send. FrankenPhone takes a new technology and reinvents it using an old one. Programmed in Python.

Optical Theremin

Contactless Instrument    •    March 2016

A contactless instrument that control pitch and volume using photodiodes. The photodiodes output different voltages based on the light levels detected, which are then sampled by a myDAQ. The data collected is then processed in LabVIEW and output to speakers or headphones as an audio signal.

RoboEtch

Etch A Sketch Drawing Robot    •    August 2015

RoboEtch analyzes images and draws them on an etch-a-sketch. We used computer vision techniques to analyze an image so that it can be translated into motor movements. RoboEtch is programmed in C++ using an arduino.

Maze Game

Random Maze Game    •    March 2013

Generates a maze using a recursive algorithm and lets you solve it. Having trouble? Maze Game already found the path and is willing to help. Programmed in Java using the Swing and AWT libraries.
